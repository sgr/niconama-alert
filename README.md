# niconama-alert.clj

[ニコニコ生放送](http://live.nicovideo.jp/) で現在放送中の番組からあなたの興味のある番組一覧を表示します。
Clojure で作成されており、 JRE (Java SE Runtime Environment) 上で動作します。

## Documentation

Please see [Project homepage](http://sgr.github.io/niconama-alert/).
